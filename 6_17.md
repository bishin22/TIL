## Git & Github

### Local에서 지워진 파일, Github에서 끌어오기

```
$ git remote add origin https://github.com/bishin22/TIL.git
```

- `remote` : 연결 / `add` : 새로운 것을 추가/ `origin` : origin 이라는 채널에

```
$ git pull origin master
```

- master branch의 git을 다 가져오기

```
$ git remote -v
```

- `remote`가 잘 되었는지 확인
